## Educational Value (0–5)

Assess how directly usable the extract is for educational purposes—specifically whether the content itself (not whether it talks about education) provides information that could be used in a primary or grade school setting.
Rate the content from 0 to 5 using the scale below:

| Score | Description |
|-------|-------------|
| 0 – No educational value | Contains no relevant educational content. Entirely unrelated to topics that could support student learning. |
| 1 – Minimal educational value | May include some facts or informative content, but it's mostly irrelevant, commercial, or superficial. |
| 2 – Low educational value | Contains fragments of useful information, but lacks clarity, reliability, or relevance to student learning. |
| 3 – Moderate educational value | Contains factual or structured information that could be useful for introductory lessons, but may require significant adaptation or filtering. |
| 4 – High educational value | Mostly clear, well-structured, and age-appropriate. Suitable for use in class or homework with minor adjustments. |
| 5 – Excellent educational value | Clear, structured, and highly relevant. Directly usable in an educational setting without modification. Ideal for lesson plans, assignments, or reading material. |

Provide a short justification (max 100 words) explaining the score.
